--- layout: default title: Buttons usage: USAGE NOTES Buttons are written to be able to be extended in many different ways to support a wide range of contexts. There should usually only be two types of button; primary and secondary. A primary button is used to highlight the most important action on the page, and generally there should only be one of these per page. If more than one primary button is required, then consider splitting the form over multiple pages. html: 
<a href="#" class="btn btn--primary">Button</a>
